# Planned Skills

This directory contains proposed Lingtu AI skills that are not implemented yet.

Do not install these directories as skills. A planned skill becomes installable only after it is moved to `packages/<skill-name>/` and includes a complete `SKILL.md`.

Use one subdirectory per planned skill. Prefer `PLAN.md` for the development plan, API notes, open questions, and acceptance criteria. Do not add `SKILL.md` until the skill is ready to be installed.
